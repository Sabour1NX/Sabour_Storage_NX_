# 随机JMcomic自动翻牌器

## ⚠️ 重要警告（必看）
**本工具不支持校园网环境使用！**
校园网通常会屏蔽目标网站域名、限制外网访问或拦截自动化脚本请求，使用时会出现「域名检测失败」「链接无法打开」「进程卡死」「帽子叔叔敲门」等问题，建议在个人宽带/手机热点环境下使用。

## 项目介绍
一款基于Python开发的自动翻牌工具，支持检测目标网站域名延迟、生成随机内容链接并自动验证跳转，内置GUI界面，操作简单。

## 核心功能
1. 多域名延迟检测：自动测试6个目标域名（如18comic.vip、jmcomic-zzz.one），筛选延迟最低的域名；
2. 随机链接生成：生成100000-999999范围内的随机ID，拼接为完整目标链接；
3. 链接验证与跳转：无头浏览器验证链接有效性，有效则调用默认浏览器打开；
4. 进程管控：支持一键启动/急停，避免脚本卡死；
5. GUI界面：简洁可视化操作，含功能按钮与图片展示。
